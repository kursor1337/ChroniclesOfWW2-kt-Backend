# ChroniclesOfWW2-kt-backend

### Backend for the Chronicles of WW2 game

### Check out Android app -> [ChroniclesOfWW2-kt-android](https://github.com/kursor1337/ChroniclesOfWW2-kt-android)

## Stack
* [Kotlin](https://kotlinlang.org/)
* [Ktor](https://ktor.io/)
* [Koin](https://insert-koin.io/)
* [KotlinX Serialization](https://kotlinlang.org/docs/serialization.html)
* [Netty](https://github.com/netty/netty)
* [HikariCP](https://github.com/brettwooldridge/HikariCP)
* [Kotlin Coroutines](https://kotlinlang.org/docs/coroutines-overview.html#documentation)
* [Exposed](https://github.com/JetBrains/Exposed)
* [PostgreSQL](https://www.postgresql.org/)
